## Field
Applied Microeconomics — with specializations in public, environmental, labor, and development economics.

- Health economics
- Crime economics
- Climate/disaster economics

## Common Data Sources
- U.S. Census / ACS (demographics, migration)
- BLS (employment, wages, CPI)
- IRS Statistics of Income (migration, county-to-county flows)
- FEMA (disaster declarations, assistance)
- SHELDUS (hazard loss data)
- NWS Storm Events Database (hurricane tracks, severity)
- FBI UCR / NIBRS (crime data)
- HMDA (mortgage data)
- NLSY / NLS (longitudinal surveys)
- EAGLE-I (power outage data)
- BEA (GDP, personal income by county)
- FRED (macro indicators)

## Common Identification Strategies
- Difference-in-Differences (DiD)
- Staggered DiD with modern estimators (CS, SA, BJS, dCDH)
- Event studies
- Instrumental Variables (IV)
- Regression Discontinuity (RD)
- Synthetic Control
- Shift-share / Bartik instruments
- Randomized experiments (rare in disaster economics)

## Field Conventions
- County-level analysis uses 5-digit FIPS codes
- Standard errors clustered at the treatment level (often county or state)
- Population-weighted regressions common for county data
- Pre-trend tests expected for any DiD design
- Bacon decomposition expected when using TWFE with staggered treatment
- Robustness to modern DiD estimators increasingly required by referees
- Event study plots with confidence intervals are standard
- Economic magnitudes expected in addition to statistical significance
